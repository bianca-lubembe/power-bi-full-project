## Project Overview
This project analyzes a global survey of data and tech professionals, uncovering insights into their roles, career paths, and the key factors that influenced their entry into the industry.

## Insights Acquired
* Data Scientists earn more salary compared to other professions
* Python is the most favorite programming language
* On a scale of 1-10 only 4.27 people are happy with their salary
* An average of 5.74 people are happy with their work/life balance
* Based on sex, Females earn more compared to men

## Tools Used
This project was entirely done on Power BI with some cleaning in the Power Query Editor
